# 🤝 Contributing to Developer Productivity Dashboard

Thank you for considering contributing to this project!

---

## 🚀 How to Contribute

1. Fork the repository
2. Clone your fork

   ```
   git clone https://github.com/your-username/dev-dashboard.git
   ```
3. Create a new branch

   ```
   git checkout -b feature/your-feature-name
   ```
4. Make your changes
5. Commit your changes

   ```
   git commit -m "Add: your feature description"
   ```
6. Push to your branch

   ```
   git push origin feature/your-feature-name
   ```
7. Open a Pull Request

---

## 📌 Contribution Guidelines

* Write clean and readable code
* Follow proper folder structure
* Add comments where necessary
* Test your code before submitting
* Keep commits meaningful

---

## 🐞 Reporting Bugs

If you find a bug:

* Clearly describe the issue
* Provide steps to reproduce
* Attach screenshots if possible

---

## 💡 Feature Requests

Feel free to suggest improvements or new features by opening an issue.

---

## 🙌 Code of Conduct

Be respectful and constructive while contributing.

---

Thanks for contributing ❤️
